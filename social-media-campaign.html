睡眠はたった30分で変えられる

当アプリは、「強制する睡眠」からの脱却を提唱します。就寝前後30分のルーティンを最適化することで、単なる睡眠時間の確保ではなく、ユーザー主導で質の高い眠りを創造。パーソナライズされたデータ分析を通じ、毎日最高の気分でスタートできるよう、眠りからサポートします。

タイプ UX design
実施期間 3日間
自分の役割 UX/UI designer
チーム構成 Alex Zhang - Background researcher Shun Fukui - UI designer Elaine Shi - Background researcher / Visual designer
(01)

課題

不規則な睡眠ルーティーンが生産性の低下を招く

「健康と生活習慣」というハッカソンのテーマをもとに、不規則な生活習慣が引き起こす睡眠の質の低下という課題を解決。就寝前後のルーティン改善で、利用者のウェルビーイング向上を目指すアプリです。

(02)

課題発見

朝ごはんを食べていない

ふと疲れていそうなメンバーたちに朝ごはんを食べたか聞いたところ、3人とも食べていないことがわかりました。そこで僕は睡眠の質が原因で時間が取れず朝ごはんが食べれていない、また睡眠の質の悪化がその日の疲れや生産性の低下を引き起こすと考えました。

そこから、私たちは睡眠の質を向上させるというテーマを掲げました。

(03)

ユーザーリサーチと分析

睡眠前後のルーティーンの重要性

①アンケート結果

  • 朝食習慣の欠如: 回答者の約23.1%が朝食を「全く食べない」と回答。さらに「週に2〜4回」まで含めると過半数近くになりました。
  • 朝食を食べない理由: 朝食を摂らない理由として、「授業に遅れる」「起きるのが遅い」「朝食を食べる時間がない」といった、起床や時間管理に関する課題が上位に挙がりました

= 朝のルーティンが睡眠から乱れていることがわかります

②睡眠の質向上

  • 寝る前30分前が命: 30分前にリラックスする準備ができないと、体と脳が「まだ起きている状態」を維持してしまい、睡眠の質を下げてしまう。
  • Orthosomnia（オルソソムニア）: これは、睡眠アプリの数値を気にしすぎて不安になりかえって眠れなくなる状態で、睡眠アプリ利用者の中で流行っています。理想的なデータと自分の誤差に過度に依存すると、ストレスや不眠を招き、睡眠を妨げていってしまいます。

= 寝る前のルーティーン・自己管理が睡眠と密接に関わっています

(04)

What is this app?

睡眠前後30分のルーティーンの最適化

(05)

ビジュアルデザイン

ミニマムで色を意識したUI

ユーザーリサーチ・睡眠関係の研究を基に、以下のビジュアルを意識しました：

1. ミニマルデザイン

就寝前後でしか使わないため、シンプルで直感的デザインを意識

2. 朝と夜の色使い

寝る前には橙色のような暖かい色、起きた時には青緑のような涼しい色が睡眠の質を最も害さないと言われているため、それに沿った色使い

3. 統一感のあるボタン

直感的に押すところがわかる色分けとハイライト

(07)

レビュー

プロジェクトの振り返りと今後の展望

3日間のハッカソンという限られた時間の中で、ユーザーリサーチからプロトタイプまで一貫したデザインプロセスを実施できました。

好き・得意と感じたところ：

  • 課題発見とソリューションの発見・デベロップメント
  • ユーザーにとって見やすい色や模様のデザイン
  • 目的志向のプロトタイピング

今後の改善点：

  • プロトタイプのUI面をさらに洗練させ、ユーザーテストを実施
  • テキストの色使い、フォント、テキスト量など、情報の見せ方を工夫する
  • シンプルさを保ちつつ、ゲーミフィケーションなどの要素を加え、ユーザーが「使いたくなる」ようなUI設計
